This directory uses the Diataxis documentation model.

| Quadrant | File | Purpose |
|----------|------|---------|
| Tutorial | [tutorial-first-demo.md](tutorial-first-demo.md) | Learn the prototype by running a first demo through Producer, Show, and Print. |
| How-to | [how-to-run-local.md](how-to-run-local.md) | Run and verify the app locally. |

| Explanation | [explanation-static-prototype.md](explanation-static-prototype.md) | Understand why the prototype is static and what should change next. |

Start with the tutorial if you are new to the project. Use the reference when
editing `src/content.js`, `src/game.js`, `src/app.js`, or `src/styles.css`.
Use the category guide, prepare-show how-to, and party runbook when preparing
the real event.
